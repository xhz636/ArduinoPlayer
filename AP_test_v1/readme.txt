版本:Beta 1.0
最后更新:2015年9月11日
描述:
本程序是作者的浙江大学暑期小学期Arduino课程作业
主要实现以下功能:
1.txt电子书的阅读、设置等
2.特定格式音频文件的播放、循环等
3.特定格式图像的显示、缩放、旋转等
4.自制游戏rom模拟器，实现SD卡中自制游戏rom的读取
5.文件处理功能
备注:
1.音频格式为FullRate@16MHz_Mono
2.图片存储数据为32位真彩色，具体见AP_tools/Arr2apimg
3.游戏编译器由于作者时间和水平有限，编写地十分简陋，只是简单实现模拟汇编代码转为机器码的过程，存在很多漏洞，恳请批评指正，作者会视具体时间和能力进行后续完善工作
